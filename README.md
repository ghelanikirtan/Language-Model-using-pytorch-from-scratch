# Transformer Language Model (from Scratch)

This project implements a **Transformer-based Language Model** using PyTorch — completely from scratch — inspired by the original paper [_Attention Is All You Need_](https://arxiv.org/abs/1706.03762).

**About:**  
A decoder-only Transformer (GPT-style) that learns to generate text token-by-token using multi-head self-attention and positional encodings.

**Network architecture content:**

- PyTorch
- Scaled Dot-Product Attention
- Multi-Head Attention
- Positional Encoding
- Feedforward Layers & LayerNorm

**Next Steps**

- Train on datasets like WikiText or OpenWebText
- Add sampling (top-k / nucleus) for text generation
- Experiment with model scaling

📄 Based on:  
Vaswani et al., _"Attention Is All You Need"_, NIPS 2017
